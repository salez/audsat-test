Aplicativo desenvolvido em Angular 16

Este aplicativo foi desenvolvido em Angular 16, utilizando o Angular CLI e o Angular Material.

  • O HTML foi desenvolvido de forma semântica e limpa, utilizando as novas tags do HTML 5 e seguindo os padrões web.
  • O CSS foi desenvolvido com SaSS ou Less, permitindo a criação de estilos reutilizáveis.
  • O aplicativo se integra com o serviço JSONPlaceholder (jsonplaceholder.typicode.com).
  • A lógica de programação orientada a objetos foi aplicada e a estruturação de módulos/componentes foi realizada no Angular.
  • O roteamento de módulos (Lazy Load) e o padrão Reactive Forms foram utilizados na estruturação do Filtro de Busca.
  • Os serviços REST são gerenciados com Observables (RxJS), permitindo um adequado gerenciamento de eventEmitters para compartilhar dados nas classes de serviços e componentes.
  • A estrutura do HTML é semântica e limpa, explorando as Tags do HTML 5 e definindo IDs, classes e atributos data- de forma inteligente.
  • O layout é responsivo e o aplicativo é otimizado para prevenir Memory Leaks.
  • O Angular Material pode ser usado para criar componentes de formulário e navegação, e também é possível definir atributos reutilizáveis com variáveis e pseudo-elementos no SaSS/Less.
  • É possível visualizar os logs armazenados e persistir os dados das ações realizadas pelo admin em REST.
  • É possível extrair e aplicar o Logo vetorialmente no projeto e aplicar CSS Animations para interações do usuário.
  • Também é possível aplicar um GRID de layout responsivo, usando Bootstrap 4 ou Angular Material.
