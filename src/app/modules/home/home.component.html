Bem-vindo ao Projeto de Teste de Desenvolvimento Audsat!

Teste desenvolvido por Rodrigo Sales Silva em 11/05/2023.

Este projeto é um teste para a vaga de desenvolvedor front-end na Audsat.

Ele consiste em uma aplicação SPA desenvolvida com Angular 16, Angular Material e RxJS.

Requisitos e Funcionalidades

  • Roteamento de módulos com carregamento lazy
  • Utilização do padrão Reactive Forms para o filtro de busca
  • Uso de Observables (RxJS) nos serviços REST
  • Estruturação semântica e limpa do HTML
  • Utilização do logo no header
  • Utilização das cores pré-estabelecidas

Alguns Diferenciais Implementados

  • Uso extensivo de componentes standalone do Angular
  • Interface desenvolvida com Angular Material
  • Animações via CSS e animations
  • Carregamento via shimmer para melhor experiência do usuário
  • Layout responsivo, seguindo a prática "mobile first"
  • Autenticação na área logada através de login

Tecnologias Utilizadas

  • Angular 16
  • Angular Material
  • RxJS
